# Contributing

Thanks for supporting this public data collection teaching repository.

## Useful Contributions

- safer and clearer parsing examples
- better compliant demo datasets
- exporter improvements
- documentation that helps learners understand the collection pipeline

## Contribution Rules

- keep examples compliant and classroom-friendly
- prefer public information and local fixtures over fragile live scraping demos
- explain why a change matters for learners, not only how it works
- update `CHANGELOG.md` for user-visible changes

## Local Checks

```bash
pip install -r requirements.txt
python examples/run_demo.py
python examples/run_api_demo.py --keyword python
```
